Certainly! Here are examples of how you can use the World Bank API to retrieve information for India in each of the mentioned categories:

1. **Country API - Information about India:**
   - Endpoint: `/countries/IN`
   - Example URL: `http://api.worldbank.org/V2/countries/IN`
   - This API will provide you with detailed information about India, including its ISO2 and ISO3 country codes, region, income level, and more.

2. **Indicator API - List of Available Indicators:**
   - Endpoint: `/indicators`
   - Example URL: `http://api.worldbank.org/V2/indicators`
   - This API will give you a list of available indicators that you can use to query specific data points.

3. **Data API - GDP Growth for India:**
   - Endpoint: `/data/NY.GDP.MKTP.CD?country=IN&date=2013:2022`
   - Example URL: `http://api.worldbank.org/V2/data/NY.GDP.MKTP.CD?country=IN&date=2013:2022`
   - This API will retrieve the Gross Domestic Product (GDP) data for India from 2013 to 2022.

4. **Topic API - List of Data Topics:**
   - Endpoint: `/topics`
   - Example URL: `http://api.worldbank.org/V2/topics`
   - This API will provide you with a list of data topics that you can explore in the World Bank dataset.

5. **Region API - Geographic Regions:**
   - Endpoint: `/regions`
   - Example URL: `http://api.worldbank.org/V2/regions`
   - This API will give you information about different geographic regions, including their codes and names.

6. **Lending Type API - Types of Lending:**
   - Endpoint: `/lendingTypes`
   - Example URL: `http://api.worldbank.org/V2/lendingTypes`
   - This API will provide information about the types of lending used by the World Bank.

7. **Income Level API - Income Categorization:**
   - Endpoint: `/incomeLevels`
   - Example URL: `http://api.worldbank.org/V2/incomeLevels`
   - This API will give you details about different income levels used to categorize countries.

8. **Source API - Data Sources:**
   - Endpoint: `/sources`
   - Example URL: `http://api.worldbank.org/V2/sources`
   - This API will provide information about the sources of data used by the World Bank.

9. **Time Series API - List of Time Series:**
   - Endpoint: `/time-series`
   - Example URL: `http://api.worldbank.org/V2/time-series`
   - This API will provide you with a list of time series available in the dataset.

These examples should help you understand how to structure API requests to retrieve various types of information related to India from the World Bank API. Remember that you can modify the parameters in the URLs to tailor the requests to your specific needs and interests.
